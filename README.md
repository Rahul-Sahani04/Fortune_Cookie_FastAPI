## API Endpoints

Here are some of the key API endpoints:

- **GET /cookie/random/**
  Get a random fortune cookie message.

- **GET /cookie/all/**
  Get all available fortune cookie messages.

- **DELETE /cookie/{message_id}/**
  Delete a specific fortune cookie message.
